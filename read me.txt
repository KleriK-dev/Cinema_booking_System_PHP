****BEFORE USING THE APPLICATION****

1. Copy and paste cinema folder to C:/xampp/htdocs

2. Go to phpMyAdmin and create a database and name it cinema.

3. Use root database user.

4. Then import the sql file that is inside the C:/xampp/htdocs/cinema/database folder.

5. Then open the application and log in as an administrator:
